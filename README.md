# Chat Top

It's a simple Socket.io based chat created for learning purposes.

## Setup

### Server

```bash
cd ./server
cp .env .env.sample
yarn install
yarn start
```

### Client

```bash
cd ./client
cp .env .env.sample
yarn install
yarn start
```

Both applications need to be running at the same time.
